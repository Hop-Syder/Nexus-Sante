## 🌟 Vision du Projet

**Nexus Santé** est une solution digitale de pointe conçue par **Nexus Partners** pour transformer l'expérience patient et optimiser la gestion médicale. Alliant esthétique premium, performances fulgurantes et accessibilité universelle, cette plateforme représente le futur des soins de santé connectés.

> "L'excellence technologique au service de la vie."

## 🚀 Fonctionnalités Clés

- **🌍 Internationalisation (i18n)** : Support complet multilingue (FR/EN) via `next-intl`.
- **🎨 Design Premium** : Interface ultra-moderne avec animations fluides (Framer Motion).
- **📱 Responsive de Précision** : Expérience utilisateur parfaite sur mobile, tablette et desktop.
- **⚡ Performance Optimale** : Score Lighthouse élevé grâce à l'architecture Next.js 16.
- **🧩 Architecture Modulaire** : Composants hautement réutilisables et maintenables.
- **📅 Gestion de Contenu** : Sections dynamiques (Services, Équipe, FAQ, Témoignages).

## 🛠️ Stack Technologique

### Frontend

- **Framework** : [Next.js 16+](https://nextjs.org/) (App Router)
- **Library** : [React 19](https://react.dev/)
- **Styling** : [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations** : [Framer Motion](https://www.framer.com/motion/)
- **Sliders** : [Swiper.js](https://swiperjs.com/)

### Outils & Qualité

- **Langage** : [TypeScript](https://www.typescriptlang.org/)
- **I18n** : [next-intl](https://next-intl-docs.vercel.app/)
- **Linting** : ESLint
- **Type Checking** : TypeScript Strict Mode

## 📂 Structure du Projet

```text
src/
├── app/               # Routes et Layouts (App Router)
│   └── [locale]/      # Gestion des langues
├── components/        # Composants UI réutilisables
│   ├── layout/        # Navbar, Footer
│   ├── sections/      # Sections de la page d'accueil
│   └── ui/            # Atomes graphiques (boutons, inputs)
├── context/           # Contextes React (Theme, Auth, etc.)
├── i18n/              # Configuration de l'internationalisation
├── messages/          # Dictionnaires de traduction (JSON)
└── public/            # Assets statiques (Images, Icons)
```

## 🛠️ Installation et Démarrage

### Prérequis

- Node.js 20.x ou supérieur
- npm / pnpm / yarn

### Étapes

1. **Cloner le projet**

   ```bash
   git clone https://github.com/nexus-partners/nexus-sante.git
   cd nexus-sante
   ```

2. **Installer les dépendances**

   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**

   ```bash
   npm run dev
   ```

4. **Build pour la production**

   ```bash
   npm run build
   ```
